## Usage Notes

Before using any source or firmware dump from this repository:

1. Confirm the exact device codename.
2. Check that the firmware version matches your target build.
3. Backup your existing partitions where possible.
4. Read device-specific flashing instructions carefully.
5. Do not flash firmware intended for another variant or region.
